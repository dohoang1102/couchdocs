A list of CouchDB related projects, clients and libraries.

Clients

  • Couchapp Command line tool to manage standalone CouchDB applications and send/clone documents from/to CouchDB.
  • CouchDBX â€” The one-click CouchDB package for the Mac
  • Davenport: Davenport is a plugin container for building native Mac OS X applications
  • CouchDB-FUSE: mount document attachments on a virtual filesystem

  • Fuschia is a graphical document browser for CouchDB

  • Out of date

    • Levitz - XUL based CouchDb utility client
    • Valance GUI client in PyGTK

Libraries

  • JavaScript libraries
    • CouchDB's built-in synchronous JS library for the test suite
    • CouchDB's built-in asynchronous jQuery plugin
    • jQuery CouchDB Library

  • Erlang interfaces:

    • Erlang CouchDB Kit using Erlang messages instead of HTTP
    • Erlang interface to CouchDB
    • erlang_couchdb
    • eCouch

  • Java interfaces:

    • Ektorp CouchDB connector
    • CouchDB4J Java bindings

  • Scala driver and View Server:

    • scouchdb Scala bindings

  • Perl interfaces:

    • Net::CouchDb
    • CouchDB::Client
    • POE::Component::Client::CouchDB

  • Perl tools:

    • CouchDB::View, handling Perl views on both the client and server sides
    • CouchDB::Deploy, simple configuration to help deploy applications that use CouchDB

  • PHP libraries

    • PHPillow, an object orientated wrapper for CouchDB.
    • PHPCouch, a CouchDB library for PHP
    • PHP library for CouchDb
    • PHP extension for CouchDB

  • Ruby libraries

    • CouchRest The base library for most Ruby CouchDB libraries
    • CouchPotato integrates Rails and CouchDB
    • CouchFoo (ActiveRecord matching API to CouchDB)
    • CouchObject (Ruby client + JsServer for views in Ruby)
  • Python libraries
    • CouchDB Python Library
    • Couchdbkit, a CouchDB framework for Python applications
    • Paisley: A Twisted Python CouchDB Client
  • Common Lisp libraries
    • CouchDB Common Lisp Library
  • Clojure libraries
    • clojure-couchdb
    • clutch
  • Squeak CouchDB Library
  • Storing_GeoData (PHP, Google Geocoding Service)

Alternatives

  • Feather DB (CouchDB clone in Java)
  • StrokeDB (A CouchDB-like database written in Ruby to make embedding into Ruby apps easier)
  • (MongoDB is a high-performance, open source, schema-free document-oriented database.)

Applications

  • See CouchDB_in_the_wild

Miscellaneous

  • CouchDBX Packaging CouchDB for Mac OS X.
  • Interactive CouchDB A CouchDB emulator/visualizer written in 100% Javascript.
  • Lounge A proxy-based partitioning/clustering framework for CouchDB.
  • Hovercraft An easy direct Erlang CouchDB library.

Full Text Searching

  • CouchDB Lucene Enables full-text searching of CouchDB documents using Lucene.
  • HyperCouch Full text indexing of CouchDB via Hyper Estraier.
  • CouchDB Solr2 Integrates full-text indexing and searching with CouchDB.
    • Per README, the CouchDB Solr2 project is no longer being maintained as of January 2009.
